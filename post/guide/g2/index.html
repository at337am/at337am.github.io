解决网页端 B 站偷跑 PCDN

PCDN 是一种内容分发网络技术，通过用户之间的 P2P（点对点）连接来共享和分发内容。BiliBili 网页端使用 PCDN 来分担服务器压力和提高内容分发效率。

PCDN 的潜在坏处：

  1. 占用带宽：PCDN 会利用用户的上传带宽，可能导致其他网络活动变慢。
  2. 安全隐患：由于涉及 P2P 连接，可能带来安全和隐私风险。
  3. 不稳定的观看体验：PCDN 的性能依赖于其他用户的上传速度，可能导致视频加载速度和质量不稳定。

防止 WebRTC 泄露真实 IP 地址

WebRTC 是一种允许网页浏览器进行实时语音、视频通信和 P2P 文件共享的技术。它通过浏览器内置的 API 实现，不需要额外插件。

WebRTC 的潜在坏处：

  1. IP 地址泄露：WebRTC 可以绕过传统的隐私保护措施，直接暴露用户的真实 IP 地址，即使在使用 VPN 时也可能被泄露。
  2. 隐私风险：通过获取用户的真实 IP 地址，网站和恶意攻击者可以追踪用户的真实位置，降低隐私保护。
  3. 安全漏洞：WebRTC 的实时通信特性可能被利用进行网络攻击，尤其是在配置不当或未加密的情况下。
