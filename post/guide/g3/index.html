RustDesk 是一款远程访问和远程控制软件，主要由 Rust 编写，可以远程维护计算机和其他设备。
适用于 Windows、macOS、iOS、Android 和 Linux 等操作系统。 via Wiki

docker-compose.yml

networks:
  rustdesk-net:
    external: false

services:
  hbbs:
    container_name: hbbs
    ports:
      - 21115:21115
      - 21116:21116
      - 21116:21116/udp
      - 21118:21118
    image: rustdesk/rustdesk-server:1.1.10-3
    command: hbbs -r 公网ip地址:21117
    volumes:
      - ./data:/root
    networks:
      - rustdesk-net
    depends_on:
      - hbbr
    restart: unless-stopped

  hbbr:
    container_name: hbbr
    ports:
      - 21117:21117
      - 21119:21119
    image: rustdesk/rustdesk-server:latest
    command: hbbr
    volumes:
      - ./data:/root
    networks:
      - rustdesk-net
    restart: unless-stopped

服务启动后，在 RustDesk 中设置，填入服务器 IP

本人嫌麻烦没有设置 key，详细教程 RustDesk 文档

云服务器的入站规则需要开启 udp 端口和 tcp 端口：21115-21119
